HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menyusun konten di halaman web.

CSS

CSS merupakan singkatan dari Cascading Style Sheets, CSS digunakan untuk menentukan gaya dari tampilan website yang akan kamu buat seperti tata letak halaman, warna , dan font.

PHP VANILLA

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis.

JAVASCRIPT

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif.

BOOTSTRAP-5

Bootstrap adalah framework front-end yang dirancang untuk membantu pengembang web dalam membuat desain web responsif dan mobile-first dengan cepat dan efisien.

Laravel 10

Laravel adalah salah satu framework PHP yang sangat populer untuk pengembangan aplikasi web. Laravel dirancang untuk memudahkan pengembang dalam membangun aplikasi web dengan cepat dan efisien sambil tetap menjaga struktur dan kebersihan kode.

Microsoft Word

Microsoft Word atau MS Word adalah program aplikasi pengolah kata (word processor) yang dikembangkan oleh Microsoft.

Microsoft Excel

Microsoft Excel adalah aplikasi perangkat lunak spreadsheet yang dikembangkan oleh Microsoft, digunakan untuk mengelola, menganalisis, dan memvisualisasikan data dalam bentuk tabel.

Figma

Figma adalah aplikasi desain berbasis web yang dapat digunakan untuk membuat, berkolaborasi, dan membuat prototipe produk digital.

Fotografi

Fotografi adalah seni, ilmu, dan praktik mengambil gambar dengan menggunakan cahaya.
